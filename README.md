## 疑难解答

**Q：在晚上总是断开链接？**  
A：请检查手机是否有如「夜间休眠省电」的设置，请将其关闭

**Q：开启了「铃声 + 悬浮窗」，但没有铃声？**   
A：请检查手机铃声音量是否开启，请检查是否授予悬浮窗权限（* 在当前版本中，若检测到无悬浮窗权限，则不会响铃）

**Q：悬浮窗权限经常丢失？**  
A：请前往手机设置，找到类似于「应用设置」-> 「应用管理」-> 找到本应用 -> 「权限管理」-> 「更多权限」-> 打开「后台弹出页面」与「显示悬浮窗」权限（* 以 MIUI 为例）
